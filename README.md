# Single Page HTML CV Project

## 📌 Project Description
This is a simple single-page CV (resume) built using HTML.  
The project focuses on using semantic HTML tags, meta tags, and basic structure.

---

## 🚀 Features
- Semantic HTML tags (header, section, article, aside, footer)
- Meta tags for SEO and responsiveness
- Open Graph (OG) tags for link preview
- Structured CV layout

---

## 🛠️ Technologies Used
- HTML5
